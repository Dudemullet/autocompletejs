Options Object

Property Type Required Default Description
initialList String yes n/a

The initialList property should be the first list started when the user clicks on the autocomplete component.
lists Object of lists yes n/a

The lists property contains all of the lists used to power the dropdown options.

Please see the List Object reference below for more information on List Objects.

List Object

Property Type Required Default Description
allowFreeform Boolean no false

If true, allows the user to enter free-form text.

If false, the user must select a value from the list.

There can still be list values when allowFreeform is set to true, but the user does not have to select one.
maxResults Number
or
false		 no false

postProcess is an optional function you can include to process data that is returned from the server.
noResults String
or
Function		 no 'No results found'

An HTML string to display when allowFreeform is false and there are no results to show.

If noResults is a function, the first argument is the current value of the search bar and the second argument is the text that the user has entered.

The function should return an HTML string which will be displayed in the dropdown list (but not be selectable as an option).

Note: noResults does not get HTML escaped when inserted into the DOM.
postProcess Function no n/a

postProcess is an optional function you can include to process data that is returned from the server.
preProcess Function no n/a

preProcess is an optional function you can include to process data before it is sent to the server.
query String
or
Function		 no 'q={value}'

query is an optional function you can include to process data before it is sent to the server.
searching String
or
Function		 no 'Searching...'

An HTML string to display when searching for results.

If searching is a function, the first argument is the current value of the search bar and the second argument is the text that the user has entered.

The function should return an HTML string which will be displayed in the dropdown list (but not be selectable as an option).

Note: searching does not get HTML escaped when inserted into the DOM.
url String no n/a

postProcess is an optional function you can include to process data that is returned from the server.
