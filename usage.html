Options Object

Property Type Default Required Description
initialList string
Should be the property name of a list in the lists property.		 n/a yes The initialList property should be the first list started when the user clicks on the autocomplete component.

List Object

Property Type Default Required Description
allowFreeform boolean false no

If true, allows the user to enter free-form text. If false, the user must select a value from the list.

There can still be list values when allowFreeform is set to true, but the user does not have to select one.
